#include <vector>
#include <numeric>
#include <cmath>

double volatility(const std::vector<double>& r) {
  double mean = std::accumulate(r.begin(), r.end(), 0.0) / r.size();
  double sum = 0.0;

  for (double x : r) {
    sum += (x - mean) * (x - mean);
  }

  return std::sqrt(sum / (r.size() - 1));
}

C++ core concepts

What you need to know to actually code in C++

C++ rewards precision. You need to understand the language, the compiler, memory ownership, and how to write code that is both fast and safe.

C++ is compiled before it runs. You usually write .cpp files, include headers, compile with a compiler, then link into an executable.

g++ -std=c++20 main.cpp -O2 -Wall -o quant_app
./quant_app

Finance calculations usually use double, not float. Avoid integer division mistakes.

double price = 101.25;
double return_ = (105.00 - price) / price;

Use references for clean parameter passing, smart pointers for ownership, and avoid manual new/delete unless you truly need them.

double mean(const std::vector<double>& values);

Use std::vector, std::array, std::map, std::unordered_map, and algorithms like accumulate, sort, and transform.

std::sort(returns.begin(), returns.end());

Use classes to represent instruments, portfolios, pricing models, simulations, and data feeds.

class Option {
public:
  double spot, strike, rate, vol, maturity;
};
